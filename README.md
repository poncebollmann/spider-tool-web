# SPIDER Protocol Tool — Web

Versión web de la herramienta SPIDER Protocol Tool, construida con **Spring Boot** y **Thymeleaf**.

## ¿Qué hace?

Permite a investigadores construir estrategias de búsqueda booleana siguiendo el protocolo SPIDER directamente desde el navegador, sin instalación.

## Tecnologías

- Java 21
- Spring Boot 3.2
- Thymeleaf
- Maven

## Cómo ejecutarlo

```bash
./mvnw spring-boot:run
```

Abre el navegador en **http://localhost:8080**
